
Time calibration README
=======================

fossildat.csv: fossil constraints used for time calibration
MCMCTreeAnalyses: input, output and control files for MCMCTree analyses
primarytree-imputed-trees: set of trees with imputed taxa, used for diversification analyses
treePL_mainTree: intermediate files for time calibration treePL analyses
pseudoposterior-imputed-trees: taxonomic imputation on trees in the pseudo-posterior
