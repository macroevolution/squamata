
Supplemental figures README
===========================

phylogenomic_samples.csv: metadata associated with phylogenomic samples, elements are used for figures S1 and S2. 
SupplInfo_FigS1.drop_loci_lnl.csv: intermediate file for figure S1, lists loci that were excluded. 
SupplInfo_FigS1.sample_matrix.csv: intermediate file for figure S1, presence/absence matrix of samples and loci. 
SupplInfo_FigS33.phyparts.tre: intermediate file for figure S33. 
SupplInfo_FigS38.previous_phylogenetic_studies.xlsx: Intermediate file summarizing information that is shown in figure S38.
